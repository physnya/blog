Lesson 3 热力学关系

约 1179 字大约 4 分钟

2026-03-05

可逆 Carnot 机效率只与热源温度有关，与工作物质无关. 为此要证明：

η=1−Q2Q1,Q2Q1=F(θ1,θ2)\eta = 1- \frac{Q_2}{Q_1},\quad \frac{Q_2}{Q_1} = F(\theta_1,\theta_2) η=1−Q1​Q2​​,Q1​Q2​​=F(θ1​,θ2​)

引入两个辅助可逆 Carnot 机，其中一个用 θ2\theta_2θ2​ 作为高温热源，θ3\theta_3θ3​ 作为低温热源，输出 WBW_BWB​；第三个热机是反向运行，输入功为 WA+WBW_A+W_BWA​+WB​，两热源为 θ1,θ3\theta_1,\theta_3θ1​,θ3​. 因此

F(θ1,θ2)=Q2Q1=Q2/Q3Q1/Q3=Q3/Q1Q3/Q2=F(θ1,θ3)F(θ2,θ3)F(\theta_1,\theta_2)=\frac{Q_2}{Q_1}=\frac{Q_2/Q_3}{Q_1/Q_3} = \frac{Q_3/Q_1}{Q_3/Q_2} = \frac{F(\theta_1,\theta_3)}{F(\theta_2,\theta_3)} F(θ1​,θ2​)=Q1​Q2​​=Q1​/Q3​Q2​/Q3​​=Q3​/Q2​Q3​/Q1​​=F(θ2​,θ3​)F(θ1​,θ3​)​

这个系统和 θ3\theta_3θ3​ 应该没关系，所以函数应该是正比例的形式，由此可以定义绝对温标.

Clausius 不等式：

∑i in a circleQiTi⩽0, or continuous version ∮dQT⩽0\sum_{i\text{ in a circle}}\frac{Q_i}{T_i}\leqslant 0,\text{ or continuous version }\oint\frac{\text{d}Q}{T} \leqslant 0 i in a circle∑​Ti​Qi​​⩽0, or continuous version ∮TdQ​⩽0

其中 QiQ_iQi​ 全部代表的是吸热 (吸热为正). 上式取等，当且仅当过程可逆.

/Proof/

利用 Carnot 定理，

η=1−Q2Q1⩽1−T2T1\eta = 1-\frac{Q_2}{Q_1} \leqslant 1-\frac{T_2}{T_1} η=1−Q1​Q2​​⩽1−T1​T2​​

注意这里的 Q2Q_2Q2​ 应该改写为 −Q2-Q_2−Q2​ (吸热为正)，得到

Q1T1+Q2T2⩽0\frac{Q_1}{T_1}+\frac{Q_2}{T_2}\leqslant0 T1​Q1​​+T2​Q2​​⩽0

之后推广到多个热源的情况就行了.

在 Clausius 不等式之后，自然可以引入熵的概念. 考虑一个可逆过程，有两种不同的路径 R1R_1R1​ 和 R2R_2R2​，有

∫if(R1)dQT=∫if(R2)dQT\underset{(R_1)}{\int_i^f}\frac{\text{d}Q}{T} = \underset{(R_2)}{\int_i^f}\frac{\text{d}Q}{T} (R1​)∫if​​TdQ​=(R2​)∫if​​TdQ​

某个量 dQ/T=dS\text{d}Q/T = \text{d}SdQ/T=dS 的积分和路径无关，因此它必须是某个态函数的微分，就有

Sf−Si=∫ifdQTS_f-S_i = \int_i^f\frac{\text{d}Q}{T} Sf​−Si​=∫if​TdQ​

提示

「熵」这个字是胡刚复教授之前在 Planck 教授来中国访问，为他做翻译时临场造出来的字.

到此为止，我们可以对可逆、准静态的过程写出一个 热力学基本关系：

dU=TdS+∑iYidyi\text{d}U = T\text{d}S + \sum_iY_i\text{d}y_i dU=TdS+i∑​Yi​dyi​

对于一个可逆过程 R1R_1R1​ 和一个不可逆过程 R2R_2R2​，

∫if(R1)dQT⩽∫if(R2)dQT\underset{(R_1)}{\int_i^f}\frac{\text{d}Q}{T} \leqslant \underset{(R_2)}{\int_i^f}\frac{\text{d}Q}{T} (R1​)∫if​​TdQ​⩽(R2​)∫if​​TdQ​

它们构成的循环是一个不可逆过程，这个过程造成了熵增 (也就是上式两边移项一下)，如果是绝热的过程，那么直接可以得到 dS⩾0\text{d}S\geqslant0dS⩾0，这就是熵增原理. 一般我们表述时，会针对孤立系统来说，因为孤立系统都是绝热的过程 —— 在这个意义上可以说 孤立系统的熵不减，这是热力学第二定律的第三种表述 (Kelvin 表述).

/Example/

质量相同、温度分别为 T1,T2T_1,T_2T1​,T2​ 的两杯水等压绝热混合，求末态熵变.

末态温度为 (T1+T2)/2(T_1+T_2)/2(T1​+T2​)/2. 用 T,pT,pT,p 作为变量，

dS=dU+pdVT=dHT=CpdTT\text{d}S = \frac{\text{d}U+p\text{d}V}{T} = \frac{\text{d}H}{T} = \frac{C_p\text{d}T}{T} dS=TdU+pdV​=TdH​=TCp​dT​

积分之后得到两杯水分别的熵变，相加

ΔS=Cp(ln⁡T1+T22T1+ln⁡T1+T22T2)\Delta S = C_p\left(\ln\frac{T_1+T_2}{2T_1}+\ln\frac{T_1+T_2}{2T_2}\right) ΔS=Cp​(ln2T1​T1​+T2​​+ln2T2​T1​+T2​​)

对于非绝热过程进行的方向判断，引入两个量，Helmholtz 自由能 F=U−TSF=U-TSF=U−TS 和 Gibbs 自由能 G=U−TS+pVG=U-TS+pVG=U−TS+pV.

由熵增原理，可证对于等 TTT 等 VVV 过程，ΔF⩽0\Delta F\leqslant0ΔF⩽0；对于等 TTT 等 ppp 过程，ΔG⩽0\Delta G\leqslant0ΔG⩽0.

特性函数：适当选取自变量，只需一个热力学量就可决定均匀系统的全部热力学性质，这样的函数称为特性函数. 其中最常用的四个是

  • U=U(S,V)U=U(S,V)U=U(S,V)
  • H=H(S,p)H=H(S,p)H=H(S,p)
  • F=F(T,V)F=F(T,V)F=F(T,V)
  • G=G(T,p)G=G(T,p)G=G(T,p)

很多时候会通过 Legendre 变换 ydx=d(xy)−xdyy\text{d}x=\text{d}(xy)-x\text{d}yydx=d(xy)−xdy 的方式来切换自变量，使得我们能够做比较方便的计算. 同时利用上述四个热力学基本关系的变体，可以推导出 Maxwell 关系，

(∂T∂V)S=−(∂p∂S)V,(∂S∂V)T=(∂p∂T)V,(∂T∂p)S=(∂V∂S)p,(∂S∂p)T=−(∂V∂T)p\begin{aligned} \left(\frac{\partial T}{\partial V}\right)_S=-\left(\frac{\partial p}{\partial S}\right)_V,&\quad \left(\frac{\partial S}{\partial V}\right)_T=\left(\frac{\partial p}{\partial T}\right)_V,\\\\ \quad\left(\frac{\partial T}{\partial p}\right)_S=\left(\frac{\partial V}{\partial S}\right)_p,&\quad\left(\frac{\partial S}{\partial p}\right)_T=-\left(\frac{\partial V}{\partial T}\right)_p \end{aligned} (∂V∂T​)S​=−(∂S∂p​)V​,(∂p∂T​)S​=(∂S∂V​)p​,​(∂V∂S​)T​=(∂T∂p​)V​,(∂p∂S​)T​=−(∂T∂V​)p​​

因为推导时用到了二阶偏导数交换顺序，所以有个要求是二阶导数连续，否则数学上不成立 —— 换成物理来说，就是不能出现相变.

熵的标准全微分 (直接用可观测量表示的熵)，以 T,VT,VT,V 为例，

dS=(∂S∂T)VdT+(∂S∂V)TdV=CVTdT+(∂p∂T)VdV\text{d}S =\left(\frac{\partial S}{\partial T}\right)_V\text{d}T+\left(\frac{\partial S}{\partial V}\right)_T\text{d}V = \frac{C_V}{T}\text{d}T+\left(\frac{\partial p}{\partial T}\right)_V\text{d}V dS=(∂T∂S​)V​dT+(∂V∂S​)T​dV=TCV​​dT+(∂T∂p​)V​dV

对于内能，类似地有能态方程，

dU=CVdT+[(∂p∂T)VT−p]dV\text{d}U = C_V\text{d}T + \left[\left(\frac{\partial p}{\partial T}\right)_VT-p \right]\text{d}V dU=CV​dT+[(∂T∂p​)V​T−p]dV

其中第二项还可以化简，有

Cp−CV=−T(∂p∂T)V2(∂V∂p)TC_p-C_V = -T\left(\frac{\partial p}{\partial T}\right)_V^2\left(\frac{\partial V}{\partial p}\right)_T Cp​−CV​=−T(∂T∂p​)V2​(∂p∂V​)T​
