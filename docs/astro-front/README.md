## README

这里是 [physnya](/) 的「天体力学前沿讲座」笔记，是在上这门课程时，在课堂上记下的笔记. 因为是边上课边记笔记，难免出现纰漏，还请多多指正.

虽然名为「前沿讲座」，但是因为是任选课，难度并不高，甚至比 [星系与宇宙](/cosmos/) 简单不少. 但是课程每节课都会请不同的教授前来授课，用比较通俗的语言讲述自己领域的前沿内容.

这门课是全英文授课，上课也必须使用英文讨论问题.
