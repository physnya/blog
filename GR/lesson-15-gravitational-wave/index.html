Lesson 15 引力波

约 1435 字大约 5 分钟

2026-04-16

为了计算，首先我们要对 Einstein 方程做 linear expansion. 这件事情有两个特点：简单，这是为数不多的 Einstein 能亲自解决的问题；同时这是一个 weakly interacting 的系统.

Newton 的引力方程没有对时间的导数，是静态的；但是 Einstein 方程中存在对时间的二阶导数这样的内容，我们称之为有 dynamical freedom，这表征了引力波存在的可能性.

第一步 gμν=ημν+hμνg_{\mu\nu}=\eta_{\mu\nu}+h_{\mu\nu}gμν​=ημν​+hμν​. 算联络：

Γμλρ=12gμν(gνλ,ρ+gνρ,λ−gρλ,ν)≈12ημν(hνλ,ρ+hνρ,λ−hρλ,ν)\begin{aligned} \Gamma^\mu{}_{\lambda\rho} &= \frac{1}{2}g^{\mu\nu}(g_{\nu\lambda,\rho}+g_{\nu\rho,\lambda}-g_{\rho\lambda,\nu})\\\\ &\approx\frac{1}{2}\eta^{\mu\nu}(h_{\nu\lambda,\rho}+h_{\nu\rho,\lambda}-h_{\rho\lambda,\nu}) \end{aligned} Γμλρ​​=21​gμν(gνλ,ρ​+gνρ,λ​−gρλ,ν​)≈21​ημν(hνλ,ρ​+hνρ,λ​−hρλ,ν​)​

Riemann 张量的后面两项有 Γ×Γ\Gamma\times\GammaΓ×Γ 这样的内容，但是 Γ\GammaΓ 本来就是 hhh 的一阶量，因此可以忽略这两项，Riemann 张量只剩下前面两项导数项. Ricci 张量因此变成

Rμν=∂Γλλμ∂xν−∂Γλμν∂xλ+O(h2)≈12(□hμν−∂2∂xλ∂xμhλν−∂2∂xλ∂xνhλμ+∂2∂xμ∂xνhλλ)\begin{aligned} R_{\mu\nu} &= \frac{\partial\Gamma^\lambda{}_{\lambda\mu}}{\partial x^\nu} - \frac{\partial\Gamma^\lambda{}_{\mu\nu}}{\partial x^\lambda} + \mathcal{O}(h^2)\\\\ &\approx \frac{1}{2}\left(\Box h_{\mu\nu} - \frac{\partial^2}{\partial x^\lambda\partial x^\mu}h^\lambda{}_\nu - \frac{\partial^2}{\partial x^\lambda\partial x^\nu}h^\lambda{}_\mu + \frac{\partial^2}{\partial x^\mu\partial x^\nu}h^\lambda{}_\lambda\right) \end{aligned} Rμν​​=∂xν∂Γλλμ​​−∂xλ∂Γλμν​​+O(h2)≈21​(□hμν​−∂xλ∂xμ∂2​hλν​−∂xλ∂xν∂2​hλμ​+∂xμ∂xν∂2​hλλ​)​

其中 □\Box□ 是 d'Alembert 算符. 利用 Einstein 方程的另一形式 Rμν=−8πG(Tμν−12ημνT)\displaystyle{R_{\mu\nu}=-8\pi G\left(T_{\mu\nu}-\frac{1}{2}\eta_{\mu\nu}T\right)}Rμν​=−8πG(Tμν​−21​ημν​T) (这里已经一阶近似了)，得到所谓 linearized Einstein equation

□hμν−∂2∂xλ∂xμhλν−∂2∂xλ∂xνhλμ+∂2∂xμ∂xνhλλ=−16πGSμν\Box h_{\mu\nu} - \frac{\partial^2}{\partial x^\lambda\partial x^\mu}h^\lambda{}_\nu - \frac{\partial^2}{\partial x^\lambda\partial x^\nu}h^\lambda{}_\mu + \frac{\partial^2}{\partial x^\mu\partial x^\nu}h^\lambda{}_\lambda = -16\pi GS_{\mu\nu} □hμν​−∂xλ∂xμ∂2​hλν​−∂xλ∂xν∂2​hλμ​+∂xμ∂xν∂2​hλλ​=−16πGSμν​

Sμν(1)=Tμν−12ημνTS_{\mu\nu}^{(1)} = T_{\mu\nu}-\frac{1}{2}\eta_{\mu\nu}T Sμν(1)​=Tμν​−21​ημν​T

另外，可以取某一个坐标使得 gμνΓλμν=0g^{\mu\nu}\Gamma^\lambda{}_{\mu\nu} = 0gμνΓλμν​=0. 一阶下，这里用 η\etaη，可以规定 hhh 的坐标条件

∂∂xμhμν=12∂∂xνhμμ\frac{\partial}{\partial x^\mu} h^\mu{}_\nu = \frac{1}{2}\frac{\partial}{\partial x^\nu}h^\mu{}_\mu ∂xμ∂​hμν​=21​∂xν∂​hμμ​

在坐标变换下，

hμν′=hμν−εν,μ−εμ,ν,x′μ=xμ+εμ(x)h'_{\mu\nu} = h_{\mu\nu} - \varepsilon_{\nu,\mu} - \varepsilon_{\mu,\nu},\qquad x'^\mu = x^\mu+\varepsilon^\mu(x) hμν′​=hμν​−εν,μ​−εμ,ν​,x′μ=xμ+εμ(x)

升降指标也变为 η\etaη 的升降，

δμρ=gμνgνρ=(ημν+h~μν)(ηνρ+hνρ)=δμρ+h~μνηνρ+ημνhνρ+O(h2)\delta^\mu{}_\rho = g^{\mu\nu}g_{\nu\rho}= (\eta^{\mu\nu}+\tilde h^{\mu\nu})(\eta_{\nu\rho}+h_{\nu\rho}) = \delta^\mu{}_\rho + \tilde h^{\mu\nu}\eta_{\nu\rho}+\eta^{\mu\nu}h_{\nu\rho}+\mathcal{O}(h^2) δμρ​=gμνgνρ​=(ημν+h~μν)(ηνρ​+hνρ​)=δμρ​+h~μνηνρ​+ημνhνρ​+O(h2)

得到 h~μν=−ημνhνρηρν=−hμν\tilde h^{\mu\nu} = -\eta^{\mu\nu}h_{\nu\rho}\eta^{\rho\nu} = -h^{\mu\nu}h~μν=−ημνhνρ​ηρν=−hμν. 对于混合指标的情况，同理计算得到

∂∂x′μh′μν=∂∂xμhμν−εμ,μ,ν−□εν12∂∂x′νh′μμ=12∂∂xνhμμ−εμ,μ,ν\begin{aligned} &\frac{\partial}{\partial x'^\mu}h'^\mu{}_\nu = \frac{\partial}{\partial x^\mu}h^\mu{}_\nu - \varepsilon^\mu{}_{,\mu,\nu}-\Box\varepsilon_\nu\\\\ &\frac{1}{2}\frac{\partial}{\partial x'^\nu}h'^\mu{}_\mu = \frac{1}{2}\frac{\partial}{\partial x^\nu}h^\mu{}_\mu - \varepsilon^\mu{}_{,\mu,\nu} \end{aligned} ​∂x′μ∂​h′μν​=∂xμ∂​hμν​−εμ,μ,ν​−□εν​21​∂x′ν∂​h′μμ​=21​∂xν∂​hμμ​−εμ,μ,ν​​

如果要使得 LHS 两者相等，那么一定可以解出一个特定的 ε\varepsilonε (坐标变换) 实现这个要求. 所以说这个坐标条件是合理的. 取定这个坐标条件之后，之前 hhh 的方程就变为

{□hμν=−16πGSμν∂∂xμhμν=12∂∂xνhμμ\left\{\begin{aligned} &\Box h_{\mu\nu} = -16\pi GS_{\mu\nu}\\\\ &\frac{\partial}{\partial x^\mu} h^\mu{}_\nu = \frac{1}{2}\frac{\partial}{\partial x^\nu}h^\mu{}_\mu \end{aligned}\right. ⎩⎨⎧​​□hμν​=−16πGSμν​∂xμ∂​hμν​=21​∂xν∂​hμμ​​

其中第一个就是简单的 d'Alembert 方程，我们早已解过无数遍. 用 Green 函数，

□G(t−t′;x⃗−x⃗′)=δ(t−t′)δ3(x⃗−x⃗′),□=∇2−∂2∂t2\Box G(t-t';\vec{x}-\vec{x}') = \delta(t-t')\delta^3(\vec{x}-\vec{x}'),\qquad\Box = \nabla^2 -\frac{\partial^2}{\partial t^2} □G(t−t′;x−x′)=δ(t−t′)δ3(x−x′),□=∇2−∂t2∂2​

解得

G(t−t′;x⃗−x⃗′)=−14π∣x⃗−x⃗′∣δ3(∣x⃗−x⃗′∣−(t−t′))G(t-t';\vec{x}-\vec{x}') = -\frac{1}{4\pi|\vec{x}-\vec{x}'|}\delta^3(|\vec{x}-\vec{x}'|-(t-t')) G(t−t′;x−x′)=−4π∣x−x′∣1​δ3(∣x−x′∣−(t−t′))

代进去做逆 Fourier 变换，

hμν(x⃗,t)=∫d3x⃗′dt′(−16πGSμν(x⃗′,t′))G(t−t′;x⃗−x⃗′)=4G∫d3x⃗′Sμν(x⃗′,t−∣x⃗−x⃗′∣)∣x⃗−x⃗′∣\begin{aligned} h_{\mu\nu}(\vec{x},t) &= \int\text{d}^3\vec{x}'\text{d}t'(-16\pi GS_{\mu\nu}(\vec{x}',t'))G(t-t';\vec{x}-\vec{x}')\\\\ &= 4G\int\text{d}^3\vec{x}'\frac{S_{\mu\nu}(\vec{x}',t-|\vec{x}-\vec{x}'|)}{|\vec{x}-\vec{x}'|} \end{aligned} hμν​(x,t)​=∫d3x′dt′(−16πGSμν​(x′,t′))G(t−t′;x−x′)=4G∫d3x′∣x−x′∣Sμν​(x′,t−∣x−x′∣)​​

平面波形式是

hμν=eμνeikμxμ+eμν∗e−ikμxμh_{\mu\nu} = e_{\mu\nu}e^{\text{i}k^\mu x_\mu}+e^*_{\mu\nu}e^{-\text{i}k^\mu x_\mu} hμν​=eμν​eikμxμ​+eμν∗​e−ikμxμ​

加一个共轭项是为了变为实数.

坐标条件要求

kμeμν=12kνeμμk_\mu e^\mu{}_\nu =\frac{1}{2}k_\nu e^\mu{}_\mu kμ​eμν​=21​kν​eμμ​

同时在真空中传播，□hμν=0\Box h_{\mu\nu}=0□hμν​=0，这得到 kμkμ=0k_\mu k^\mu = 0kμ​kμ=0.

对于一个坐标变换 εμ(x)\varepsilon^\mu(x)εμ(x)，其 Fourier 变换是 ϵμ=iεμeikμxμ−iε∗μe−ikμxμ\epsilon^\mu = \text{i}\varepsilon^\mu e^{\text{i}k^\mu x_\mu}-\text{i}\varepsilon^*{}^\mu e^{-\text{i}k^\mu x_\mu}ϵμ=iεμeikμxμ​−iε∗μe−ikμxμ​，在这个变换下

eμν′=eμν+kμεν+kνεμkμe′μν=kμeμν+kνεμkμ12kνe′μμ=12kνeμμ+kν(kμεμ)\begin{aligned} &e'_{\mu\nu} = e_{\mu\nu} +k_\mu\varepsilon_\nu+k_\nu\varepsilon_\mu\\\\ &k_\mu e'^\mu{}_\nu = k_\mu e^\mu{}_\nu + k_\nu\varepsilon_\mu k^\mu\\\\ &\frac{1}{2}k_\nu e'^\mu{}_\mu = \frac{1}{2}k_\nu e^\mu{}_\mu+k_\nu(k^\mu \varepsilon_\mu) \end{aligned} ​eμν′​=eμν​+kμ​εν​+kν​εμ​kμ​e′μν​=kμ​eμν​+kν​εμ​kμ21​kν​e′μμ​=21​kν​eμμ​+kν​(kμεμ​)​

可以通过选择 εμ\varepsilon^\muεμ 的方式，让 e13,e23,e33,e00=0e_{13},e_{23},e_{33},e_{00}=0e13​,e23​,e33​,e00​=0. 再加上坐标条件给出的四个方程，只剩下下面两个不为零的自由度：

e11=−e22,e12=e21e_{11}=-e_{22},\qquad e_{12} = e_{21} e11​=−e22​,e12​=e21​

引力波用 hijTTh_{ij}^{TT}hijTT​ 描述：traceless transverse，无迹横向分量. 空间分量可以写成

hijTT=(h+h×0h×−h+0000)h_{ij}^{TT} = \begin{pmatrix} h_+&h_\times&0\\ h_\times&-h_+&0\\ 0&0&0 \end{pmatrix} hijTT​=​h+​h×​0​h×​−h+​0​000​​

也由此能够写出度规. 一般我们引入投影算符，命令单位矢量 n^=k⃗/∣k⃗∣\hat{n}=\vec{k}/|\vec{k}|n^=k/∣k∣，投影算符 Pij=δij−n^in^jP_{ij}=\delta_{ij}-\hat{n}_i\hat{n}_jPij​=δij​−n^i​n^j​. 这个算符的作用是投影到横向，也就是

Pijgjn^i=0P_{ij}g_j\hat{n}_i=0 Pij​gj​n^i​=0
